title: 'Get Article by ID'
description: 'Get single article with full content by numeric ID'

## Endpoint

<CodeGroup>
```bash cURL
curl -X GET 'https://api.ceboelha.com/insights/news/123' \
  -H 'Content-Type: application/json'
```

```javascript JavaScript
const response = await fetch('https://api.ceboelha.com/insights/news/123', {
  method: 'GET',
  headers: {
    'Content-Type': 'application/json'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get('https://api.ceboelha.com/insights/news/123')
data = response.json()
```
</CodeGroup>

## Path Parameters

<ParamField path="id" type="integer" required>
  Numeric article ID
</ParamField>

## Error Response

If the article is not found:

```json
{
  "success": false,
  "error": {
    "code": "NOT_FOUND",
    "message": "Artigo não encontrado"
  }
}
```

## Notes

- This endpoint automatically increments the view count for the article
- Only articles with `published` status are returned
- The `content` field contains Markdown that should be rendered on the client side
- No authentication is required for this endpoint
